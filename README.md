# GoogleTest Template

## Getting Started

```
$> cmake -S . -B build
$> cmake --build build
$> cd build
$build> make test
```

## Documentation

Documentation is created in Use [doxygen](https://www.doxygen.nl/). To generate
the commands, simply install it and run the `doxygen` command.
