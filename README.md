# GoogleTest Template

## Getting Started

```
$> cmake -S . -B build
$> cmake --build build
$> cd build
$build> make test
```

## Checklist

* Are the test suite names clearly demonstrating the SUT (System Under Test)?
* Have they documented the negative and positive boundary conditions?
* Do the test names clearly
